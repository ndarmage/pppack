Installation
============

PPPACK is available on PyPI, so use pip to install the package as:

.. code-block:: bash
  
  pip install pppack

To fork a development branch from GitHub, use:

.. code-block:: bash
  
  git clone git://github.com/ndarmage/pppack

The documentation can be generated by Sphinx in ``docs/``, for instance by ``make html``.
